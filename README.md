# DataBase-for-Products-inventor-PostgreSQL
A data base for products and inventory coded in C# for PostgreSQL's Data bases.

Make sure do instail libpq-dev
Update the connection string in connect_db() with your actual PostgreSQL credentials.

Bash
```
gcc database.c -o database -lpq
./database
```
Bash
```
sudo apt install libpq-dev
```

This database is ment for public use, any errors please tell me via discord located in my bio.
